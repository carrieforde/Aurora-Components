    variables

    project-prefix

    $project-prefix: '';

    Description

    Add an [optional] project prefix (or namespace) to all CSS classes.

    Type

    String

    Example

    Usage

    $project-prefix: cf;

    Output

    .cf-text-primary {}

    Usage

    <h1 class="cf-text-primary"></h1>

    color-prefixes

    $color-prefixes: (
	color: text,
	background-color: bg,
	fill: fill
);

    Description

    Prefixes for generating color classes. Leave properties on left, and only update values on right. If classes aren't needed, delete property value pair as needed.

    Type

    Map

    mixins

    pseudo-states

    @mixin pseudo-states($property, $value, $classname: '') { ... }

    Description

    Generates classes for regular & pseudo states.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $property

    A CSS property.

    		String— none
    $value

    Value to correspond with the CSS property.

    		String— none
    $classname

    The classname to user (optional). Defaults to value if not used.

    		String''

    Example

    Usage

    @include pseudo-states(font-style, italic);

    Sample Output

    .italic,
.hover\:italic:hover,
.active\:italic:active,
.focus\:italic:focus {
  font-style: italic;
}

    Sample Usage

    <a href="#" class="text-primary hover:text-secondary"></a>

    responsive

    @mixin responsive($property, $value, $classname: '') { ... }

    Description

    Generates responsive classes.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $property

    A CSS property to render.

    		String— none
    $value

    The value corresponding to the passed $property.

    		String or Number— none
    $classname

    The desired classname (optional).

    		String''

    Example

    Usage

    @include responsive(display, flex);

    Sample Output

    .sm:flex, .m:flex, .l:flex, .xl:flex { display: flex; }

    Sample Usage

    <header class="m:flex"></header>

    Used by

    • [mixin] sizes

    Author

    • Carrie Forde

    sizes

    @mixin sizes($map, $prefixes) { ... }

    Description

    Generates height and width classes.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $map

    The size map to loop through.

    		Map— none
    $prefixes

    The prefix map to loop through.

    		Map— none

    Example

    Usage

    @include sizes($widths, $width-prefixes);

    Sample Output

    .w-0 {
  width: 0;
}
.w-content {
  width: 1100px;
}
.w-full {
  width: 100%;
}
.w-screen {
  width: 100vw;
}

    Sample Usage

    <main class="max-w-full l:w-1100"></main>

    Requires

    • [mixin] responsive

    typography

    @mixin typography($map, $property, $prefix: '') { ... }

    Description

    Generates typography classes by looping through typography configuration maps.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $map

    The map to be looped through.

    		Map— none
    $property

    		String— none
    $prefix

    		String''

    Example

    Usage
